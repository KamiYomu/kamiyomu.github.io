# Family Safe Mode

Family Safe Mode is designed to hide NSFW content from websites within your KamiYomu instance.

## Overview

{: .highlight }
> **Note:** Family Safe Mode only hides content visually. Downloads will continue to process in the background.

## Enabling and Disabling

Family Safe Mode is **enabled by default**. To toggle it:

- Click the "Family Safe Mode" button in the interface
- When **green**: Family Safe Mode is enabled
- When **red**: Family Safe Mode is disabled

## Use Cases

Use Family Safe Mode to maintain a clean, professional interface free of NSFW content. This feature:

- Hides inappropriate content from view
- Does **not** delete any content
- Preserves all downloads and data
