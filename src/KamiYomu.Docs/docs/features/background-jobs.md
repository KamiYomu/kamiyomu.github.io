# Background Jobs

Background Jobs provides access to the Hangfire dashboard, a powerful tool for managing asynchronous job processing. 

## Key Features

- **Job Monitoring**: View all jobs in progress, completed, and failed states in real-time
- **Download Tracking**: Monitor the status of active download jobs
- **Job Management**: Retry failed jobs or delete jobs that are no longer needed
- **Job History**: Access detailed logs and execution history for troubleshooting
- **Recurring Jobs**: Schedule and manage recurring background tasks
- **Job Queues**: Organize jobs by priority and type using multiple queues
- **Performance Metrics**: Track job execution times and success rates

The Hangfire dashboard provides an intuitive interface to manage your background job infrastructure efficiently.
