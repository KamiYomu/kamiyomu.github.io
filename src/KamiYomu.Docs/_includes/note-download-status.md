## Viewing Download Chapter Progress

Each manga download has a status for both chapters and the overall manga.

### Download Status for Manga

| Status                | Meaning                                                                 |
|-----------------------|-------------------------------------------------------------------------|
| To Be Rescheduled     | The manga is waiting to be scheduled.                                   |
| Scheduled             | The manga download is scheduled and will start soon.                   |
| In Progress           | The manga is fetching all chapters from the source.                    |
| Completed             | The manga has finished fetching chapters.                               |
| Canceled              | The download was canceled by the user or due to reaching the maximum retry limit (10 by default). |

### Download Status for Chapters

| Status                | Meaning                                                                 |
|-----------------------|-------------------------------------------------------------------------|
| To Be Rescheduled     | The chapter is waiting to be scheduled.                                 |
| Scheduled             | The chapter download is scheduled and will start soon.                 |
| In Progress           | The chapter is being downloaded from the source.                       |
| Completed             | The chapter download has been completed.                                |
| Canceled              | The chapter download was canceled by the user or due to reaching the maximum retry limit (10 by default). |
